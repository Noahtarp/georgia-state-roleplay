# Images Directory

Place your images here:

## Required Images

### Hero Background (Homepage)
- **File:** `hero-bg.jpg`
- **Description:** Full-width background image for the homepage hero section
- **Recommended Size:** 1920x1080px (or larger, 16:9 aspect ratio)
- **Format:** JPG (optimized for web, ~200-500KB)
- **Tips:** 
  - Use a high-quality image of Georgia/roleplay scenes
  - The image will fade to black on the LEFT side where text appears
  - Choose an image with interesting detail on the RIGHT side
  - Dark/moody images work best with the fade effect

### Logo
- **File:** `logo.png`
- **Description:** Main logo for the website header
- **Recommended Size:** 200x50px (or proportional)
- **Format:** PNG (with transparency recommended)

### Application Banners
- **File:** `banner-staff.png`
- **Description:** Banner shown when user selects Staff application
- **Recommended Size:** 800x200px (or similar aspect ratio)
- **Format:** PNG

- **File:** `banner-gsp.png`
- **Description:** Banner shown when user selects GSP application
- **Recommended Size:** 800x200px (or similar aspect ratio)
- **Format:** PNG

- **File:** `banner-fbi.png`
- **Description:** Banner shown when user selects FBI application
- **Recommended Size:** 800x200px (or similar aspect ratio)
- **Format:** PNG

## Notes

- All images should be optimized for web
- Use PNG format for logos/banners with transparency if needed
- JPG for hero background (smaller file size)
- The CSS will automatically resize images to fit
- Make sure file names match exactly (case-sensitive)
